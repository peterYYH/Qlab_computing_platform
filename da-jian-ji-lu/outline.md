# Outline

大数据有许多工具，适用不同场景，实验室服务器上已经搭建可用的有以下一些：

* Hadoop
* Spark
* hive && mysql
* zeppelin
* jupyter notebook
* OpenPai
* docker
* Kubernetes

同学们可以根据自己的需求，使用不同的工具。
